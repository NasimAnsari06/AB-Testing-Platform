# 📊 Product Experimentation & A/B Testing Platform

## 📌 Project Overview

This project analyzes an A/B testing experiment conducted on a landing page to determine whether a newly designed page (Treatment Group) performs better than the existing page (Control Group) in driving user conversions.

The project follows a complete experimentation workflow used by leading product companies such as Google, Amazon, Meta, Microsoft, Airbnb, and Uber. It combines SQL, Python, Statistical Testing, and Power BI to transform raw experiment data into actionable business decisions.

---

## 🎯 Business Problem

Organizations frequently launch new product features and website designs to improve user engagement and conversions.

Before deploying changes to all users, companies must answer a critical question:

**Does the new landing page significantly improve conversion rates compared to the current page?**

This project provides a data-driven framework to answer that question through experimentation and statistical analysis.

## 📂 Dataset

Dataset: **A/B Testing Dataset (ab_data.csv)**

### Features

| Column       | Description                |
| ------------ | -------------------------- |
| user_id      | Unique user identifier     |
| timestamp    | User interaction timestamp |
| group        | Control or Treatment group |
| landing_page | Old page or New page       |
| converted    | Conversion outcome (0/1)   |

### Dataset Size

* 294,478 Records
* 5 Features
* Binary Conversion Target
